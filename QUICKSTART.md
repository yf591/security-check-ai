# 🚀 クイックスタートガイド

セキュリティチェックAIをすぐに始めるための簡単なガイドです。

## ステップ1: パッケージのインストール確認

仮想環境がアクティブになっていることを確認してください：

```bash
# 仮想環境をアクティベート（まだの場合）
source .venv/bin/activate  # Mac/Linux
# または
.venv\Scripts\activate     # Windows
```

パッケージは既にインストール済みです。

## ステップ2: アプリケーションを起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501` でアプリケーションが表示されます。

## ステップ3: サンプルデータで試す

サンプルデータ（15件のセキュリティQ&A）が `data/raw/sample_security_qa.csv` に用意されています。

1. アプリケーションのサイドバーで「🔄 data/rawから再構築」ボタンをクリック
2. データベースにサンプルデータが読み込まれます（初回は埋め込みモデルのダウンロードで数分かかります）
3. 「🔍 質問検索」タブで質問を入力して試してみてください

### 試してみる質問の例

- 「データの暗号化について教えてください」
- 「バックアップはどうなっていますか？」
- 「アクセス制御の方法は？」
- 「セキュリティ監査について」

## ステップ4: 独自のデータを追加

### 方法A: ファイルアップロード

1. サイドバーの「ファイルをアップロード」セクションを使用
2. PDF、Word、Excel、CSV、テキストファイルをアップロード
3. 「📥 ファイルを処理してDBに追加」をクリック

### 方法B: data/rawフォルダに配置

1. `data/raw/` フォルダにファイルをコピー
2. サイドバーの「🔄 data/rawから再構築」をクリック

## ステップ5: 一括質問処理

複数の質問を一度に処理したい場合：

1. 「📝 一括質問処理」タブを開く
2. テキストエリアに質問を入力（1行に1つ）、またはCSVファイルをアップロード
3. 「🚀 一括検索」をクリック
4. 結果をCSVでダウンロード可能

## トラブルシューティング

### 初回起動が遅い

初回起動時は埋め込みモデル（約90MB）のダウンロードに数分かかります。これは1回のみです。

### データベースをリセットしたい

サイドバーの「🗑️ データベースをクリア」ボタンを使用するか、以下のコマンドを実行：

```bash
rm -rf vectordb/  # Mac/Linux
rmdir /s vectordb  # Windows
```

### ポートが使用中のエラー

別のポートでアプリケーションを起動：

```bash
streamlit run app.py --server.port 8502
```
